# Project: Coffee & Cheesecake Landing Page

## The Project
A clean, modern landing page designed to showcase a café's best-selling items and encourage customers to sign up for exclusive discounts. This project focuses on a "layered" layout that balances high-quality imagery with witty, engaging storytelling.

## Key Features
* **Best Sellers Gallery:** A row of four high-quality images (Latte, Matcha, Frappe, and Cheesecake) all set to equal sizes using CSS `object-fit: cover`.
* **Witty Content:** Personality-driven descriptions for every product to create a unique brand voice.
* **Inspirational Quote:** A stylized section featuring coffee-themed quotes to establish a relaxing user experience.
* **Call to Action (CTA):** A high-visibility, bright blue "Sign Up" box designed to capture user attention and increase conversions.

## Technical Highlights
* **Flexbox Layout:** Utilized `display: flex` for horizontal alignment, vertical centering, and equal spacing across all sections.
* **Responsive Design:** Images and containers use flexible widths to ensure the site looks great on various screen sizes.
* **Clean Code:** Separation of concerns using external or internal CSS classes rather than inline styling.

## Layout Breakdown
1. **Header:** A centered title welcoming the guest.
2. **Display:** Four images with witty text captions aligned directly underneath.
3. **Quote Section:** An italicized, centered quote with the author's name.
4. **Action Layer:** A rounded blue box containing the "Sign Up" prompt and a button on the right side.
